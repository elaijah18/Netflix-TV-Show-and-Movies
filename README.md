# Netflix-TV-Show-and-Movies

### Data Set
- Listing of all TV Shows and Movies available in Netflix from 2008 - 2021 from [Netflix Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows).

### Tools 
- MySQL Workbench for data analysis - View [SQL Script] (https://github.com/elaijah18/Netflix-TV-Show-and-Movies/blob/main/Netflix%20TV%20Shows%20and%20Movies.sql).
- Tableau for data visualization - View [dashboard](https://public.tableau.com/views/NetflixMoviesandTVShows_17739799620470/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link).
